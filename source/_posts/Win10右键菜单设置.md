# 简介

Windows操作系统下自定义右键菜单功能有的时候可以实实在在的提高工作的效率

# 使用场景

## 情况一

桌面上右键弹窗里自定义

```
计算机\HKEY_CLASSES_ROOT\Directory\Background\shell
```

## 情况二

在某个文件夹上右键

```
计算机\HKEY_CLASSES_ROOT\Folder\shell\
```
