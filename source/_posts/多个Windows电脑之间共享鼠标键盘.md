# 简介

Mouse without Borders 是一款让您通过一套鼠标和键盘控制多达四台电脑的产品，让您成为电脑舰队的指挥官。这意味着，有了无边界鼠标，您可以在电脑之间复制文本或拖放文件。

# 安装

1. [下载](https://www.microsoft.com/en-us/download/details.aspx?id=35460) 

![](https://cdn.jsdelivr.net/gh/hfshaobing/picx-images-hosting@master/20250911/Snipaste_2025-09-11_09-30-47.6o3s0s0owp80.webp)

2. 安装

在两台电脑上双击下载的文件直接安装即可

3. 跳过所有设置直接进入软件设置页

![](https://cdn.jsdelivr.net/gh/hfshaobing/picx-images-hosting@master/20250911/Snipaste_2025-09-11_09-33-32.3op718gkcdq0.webp)

4. 设置Security Key

自定义一个简单的即可，注意要满16位

5. Apply(应用)

# 要点

两台电脑要在同一局域网
